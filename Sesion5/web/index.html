HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar contenido en la web. A través de etiquetas y elementos, HTML define la apariencia y la organización de una página web. Permite agregar texto, imágenes, enlaces, formularios y mucho más. Es un lenguaje de marcado fácil de aprender y es fundamental para el desarrollo de sitios web modernos y accesibles en Internet.
